Tanzania Political Trust Score Dashboard

Project Overview
The Tanzania Political Trust Score Dashboard is an interactive tool designed to analyze and visualize public trust in political figures, institutions, and policies in Tanzania. Using a combination of survey data, natural language analysis, and explainable AI, this dashboard provides insights into the political landscape and citizen sentiment.

Key Features

Interactive Visualizations: Explore trust scores across different politicians, parties, and institutions.

Explainability Layer: Understand the factors influencing each trust score with transparent AI explanations.

Data-Driven Insights: Leverage data to uncover trends in political trust over time.

User-Friendly Interface: Accessible to researchers, policymakers, journalists, and the general public.
